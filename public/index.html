Como usar
  1. Faça upload de uma imagem contendo um código de barras EAN13.
  2. Clique em "Detectar Código de Barras".
  3. O sistema tentará identificar o código de barras na imagem.
API Endpoints
  • POST /api/barcode/decode - Decodifica um código de barras de uma imagem
  • GET /api/barcode/info/:barcode - Obtém informações sobre um código de barras
